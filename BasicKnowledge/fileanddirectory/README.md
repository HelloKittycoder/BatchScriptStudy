### 文件目录操作
1. dir 显示一个目录中包含的文件目录  
显示C盘根目录下的文件  
显示C盘根目录包括子目录的文件
2. cd 切换目录  
① 默认只能在当前驱动器下切换目录  
② 若要在不同的驱动器之间切换目录，就要加上/d参数  
③ cd /   切换至当前驱动器的根目录
3. md 创建目录  
创建单个目录 md a
创建多层子目录 md a\b\c\d
4. rd 删除目录  
由于删除是非常危险的操作，这里是永久删除。默认是要用户确认的，且不允许删除非空目录  
无需确认 /q  
联通这个目录树直接删除，不管目录是否为空 /s  
rd /s /q a
5. copy 复制  
如：把当前目录下的“a.txt”文件复制到E盘  
copy a.txt E:\a.txt  
如果无需确认，可以写成 copy /y a.txt E:\a.txt
6. del 删除文件  
如：删除E盘的“a.txt”  
是永久删除，删除后无法找回  
del E:\a.txt
